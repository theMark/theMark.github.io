# Project Images

Drop project cover images here, then point the CSS at them.

**Recommended specs:** 1600×1000px, JPG, under 300KB. Use [Squoosh](https://squoosh.app) to compress.

**Suggested names:** `project-1.jpg`, `project-2.jpg`, `project-3.jpg` — to match the CSS selectors `.cover-1`, `.cover-2`, `.cover-3` in `styles.css`.
