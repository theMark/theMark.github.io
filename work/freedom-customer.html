freedom forever — customer app

Product Designer Freedom Forever Customer-facing mobile

A customer-facing mobile app for homeowners going solar with Freedom Forever. Replaced a crashing legacy app with a guided install journey, plain-language answers, and live system monitoring — designed and built end-to-end using Claude Code, the Figma MCP, and generative imagery.

context

The previous customer app was a liability. It crashed regularly, the interface was clunky, and the information it surfaced wasn't useful to the homeowners it was meant to serve. A Freedom Forever solar install is a months-long process — permitting, scheduling, install, inspection, utility activation — and during all of that, the customer is mostly waiting. Without a working app, they were waiting in the dark.

The replacement needed to do three things: walk customers through their install journey step by step, answer the questions homeowners actually ask before they have to ask them, and surface live system data once the panels were up and producing power.

my role

Sole designer and builder of the new app. After a developer set up the initial project scaffolding, I used Claude Code and the Figma MCP to design and ship production code end-to-end — branding, components, flows, copy, generated imagery, and the screens themselves.

what I did

A few of the threads:

  • Designed and built the full homeowner journey — onboarding, install milestone tracking, plain-language answers to the questions customers ask most, and live system monitoring after activation.
  • Used Claude Code as the production build environment with the Figma MCP wired in, so design and code lived in one loop with no handoff lag between mock and shipping screen.
  • Built Claude Code skills that pulled in Freedom Forever's brand colors and design tokens automatically, so generated UI stayed on-brand without manual translation.
  • Used generative imagery for every step of the install journey — visuals that turned an abstract, months-long process into something concrete the customer could see themselves inside.

outcome

Before AI-assisted development, an app at this scope would have taken the company about a year to build. With Claude Code as the design-and-build loop, it shipped in roughly one week — one person, end-to-end, production-ready.
